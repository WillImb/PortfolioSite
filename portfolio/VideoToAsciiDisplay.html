Video to Ascii

Overview

Converts the pixels of your video into Ascii text based on the brightness of each pixel. This personal project was developed with python using python libraries cv2 and pyqt during the summer of 2025
Download the .zip

Development

Brainstorming

At this time I was trying to relearn python so I could become immersed in the language again. I had not used python for a while and I figured the best way was to create a personal project.
I decided to create an video to ascii program because I wanted to create something challenging, practical, and a project I could enjoy making.

My Work

  • UI for functionality
  • Read and write files through file selector
  • Reading pixel data logic
  • Stitching modified frames into new video

I coded everything in Python. I Utilized Cv2 for video reading and frame data, and used Pyqt for the UI of the software
